# 竹梅クロス対戦評価 — 新デッキ同士のゲーム形状分離 (SOT-1668)

SOT-1665 のクロス対戦ブリッジ（梅 arena ホスト + 竹アダプタ + `agents.__path__` 合成）を
`eval/bench_cross_vs_ume.py` として本 repo に恒久化（C1）し、デッキ引数化のうえで
D1（竹: シングルプライズ速攻 `decks/deck_tempo_metal.csv`）と
D2（梅: 重量級タンク回復 `decks/deck_tank_heal_v1.csv`）の新デッキ同士を実測した。

- 実行日: 2026-07-13 / paired 先後入替 N=400 × 独立 seed（Wilson 95% CI）
- agent は両側とも各 repo の現行ルールベース（竹 `RuleBasedAgent(policy=scoring)` / 梅 `RuleAgent` R3 champion）
- 全構成で **違法出力 0 / fault 0 / draw 0 / undecided 0**（arena safety カウンタ）
- 再現手順は `eval/bench_cross_vs_ume.py` の docstring（数十秒で 1,200 試合）

## 結論

1. **ミラーデッキ構成で SOT-1665 が再現できた（統計的一致）。** 竹勝率 合算 0.445
   [0.417, 0.473]（SOT-1665: 0.458 [0.430, 0.487]、CI 重複・同一結論「梅 R3 が有意に強い」）。
   エンジンに seed API が無いため（trace 仕様 E1）、再現は bit 一致ではなく統計的一致で判定する。
2. **新デッキ同士のクロスは梅（タンク回復）が有意に強い。** 竹勝率 0.239 [0.216, 0.264]。
   先手でも 0.317 と 0.5 を大きく下回る（正直報告: 竹に有意差なし、どの席でも梅優位）。
3. **「異なる戦略」は狙ったゲーム形状の分離としてはっきり現れた（D 系列の設計意図どおり）。**
   竹の勝ちは短期決戦（平均 20.2 step）、梅の勝ちは長期戦（平均 77.4 step）で、ミラー基準
   （32.5 / 35.6 step）から両方向に分離。プライズ交換も、梅の獲得は**全て 1 枚バッチ**
   （=竹が単一プライズのみ）、竹側は 3 枚バッチ（Mega KO）がほぼ消滅（回復でタンクを
   落とせない）と、プロファイルが完全に分かれた。

### ゲーム形状の分離（受け入れ条件 3）

**step 分布** — 竹=短期決戦 / 梅=長期戦 が二峰分離:

| 構成 | 竹勝ち step (mean/median) | 梅勝ち step (mean/median) | 全体 p25–p75 |
| --- | --- | --- | --- |
| ミラー（基準） | 32.5 / 27 | 35.6 / 30 | 20–43 |
| **新デッキクロス** | **20.2 / 18** | **77.4 / 66** | 23–96（二峰化） |

竹が勝つゲームは基準より ~12 step 短く（p75=21: 勝ち筋はほぼ 21 step 以内の速攻）、
梅が勝つゲームは基準の 2 倍超に長期化（p75=117）。分布はほぼ重ならない
（竹勝ち p75=21 < 梅勝ち p25=41）。

**プライズ交換プロファイル** — 単一プライズ vs Mega の経済がそのまま観測された:

| 指標 | ミラー（基準） | 新デッキクロス |
| --- | --- | --- |
| 梅のプライズ獲得バッチ | 1枚 921 / 2枚 27 / 3枚 275 | **1枚 3,789 / それ以外 0** |
| 竹のプライズ獲得バッチ | 1枚 868 / 2枚 17 / 3枚 229 | 1枚 541 / **3枚 7**（Mega KO ほぼ不可） |
| 平均プライズ獲得（竹/梅） | 1.32 / 1.50 | 0.47 / **3.16** |
| 終局理由 prize-out : no-active | 81 : 1119 (6.8%) | **311 : 889 (25.9%)** |

- 梅の獲得が全て 1 枚バッチ = 竹デッキに複数プライズ源が無い（シングルプライズ設計どおり）。
  その結果、梅は勝つのに 6 KO が必要になり prize-out 終局が基準の 3.8 倍に増加・長期化。
- 竹の 3 枚バッチ（Mega ex KO）は 1,200 試合で 7 回のみ（ミラーでは 229 回)。回復タンクを
  落とせず、竹の速攻が間に合わない場合は取り切られる、という設計どおりの非対称。
- 敗者の到達度: 梅勝ち時の竹獲得は 0 枚が 693/913、竹勝ち時の梅獲得は 0 枚が 282/287 —
  どちらが勝つ場合も一方的で、勝ち筋が交差しない 2 つの異なるゲームが混在している。
- 注: 勝者の最終 KO 分のプライズは最終観測後に発生し得るため、勝者側の獲得数は
  「観測下限」。敗者側は正確値（ハーネス docstring 参照）。
